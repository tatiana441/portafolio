## 🌐 Cómo Publicar Tu Portafolio (Obtener Link Público)

### OPCIÓN 1: GitHub Pages (GRATIS - Recomendado)

**Ventajas:** Gratis, profesional, fácil de actualizar
**Link final:** `https://tu-usuario.github.io/portafolio`

**Pasos:**

1. Crea cuenta en GitHub (si no tienes): https://github.com/signup

2. Crea un nuevo repositorio:
   - Click en "+" arriba a la derecha → "New repository"
   - Nombre: `portafolio` (o el que quieras)
   - ✅ Marca "Public"
   - ✅ Marca "Add a README file"
   - Click "Create repository"

3. Sube los archivos:
   - Click en "uploading an existing file"
   - Arrastra: `index.html`, `styles.css`, `script.js`
   - Click "Commit changes"

4. Activa GitHub Pages:
   - Ve a Settings → Pages (menú izquierdo)
   - En "Source" selecciona "main" branch
   - Click "Save"
   - ¡Espera 1-2 minutos!

5. Tu portafolio estará en:
   ```
   https://TU-USUARIO.github.io/portafolio
   ```

**Actualizar portafolio:**
- Solo sube nuevos archivos al repositorio
- Se actualiza automáticamente en 1-2 minutos

---

### OPCIÓN 2: Netlify (GRATIS - Más Fácil)

**Ventajas:** Súper fácil, drag & drop
**Link final:** `https://tu-nombre.netlify.app`

**Pasos:**

1. Ve a: https://www.netlify.com/

2. Click "Sign up" (puedes usar email o GitHub)

3. Una vez dentro:
   - Click "Add new site" → "Deploy manually"
   - Arrastra toda la carpeta `portafolio-tatiana`
   - ¡Listo! Te da un link automáticamente

4. Personalizar link:
   - Click en "Site settings" → "Change site name"
   - Cambia a: `tatiana-munoz-bi-analyst`
   - Tu link será: `https://tatiana-munoz-bi-analyst.netlify.app`

**Actualizar portafolio:**
- Arrastra nuevos archivos en "Deploys" → "Drag and drop"

---

### OPCIÓN 3: Vercel (GRATIS - Profesional)

**Ventajas:** Rápido, profesional, analytics
**Link final:** `https://tu-nombre.vercel.app`

**Pasos:**

1. Ve a: https://vercel.com/signup

2. Regístrate con GitHub (recomendado)

3. Click "Add New" → "Project"

4. Importa tu repositorio de GitHub (del paso de GitHub Pages)
   O arrastra la carpeta si no usaste GitHub

5. Click "Deploy"

**Tu link será:** `https://portafolio-tatiana.vercel.app`

## 🎨 Personalización

### Agregar tu foto

1. Guarda tu foto profesional como `foto.jpg` en la carpeta
2. En `index.html` busca (Ctrl+F): `hero-image-placeholder`
3. Reemplaza todo el `<div class="hero-image-placeholder">` por:
   ```html
   <img src="foto.jpg" alt="Tatiana Muñoz" style="border-radius: 50%; width: 300px; height: 300px; object-fit: cover; box-shadow: 0 10px 30px rgba(0,0,0,0.3);">
   ```

### Agregar CV en PDF

1. Guarda tu CV como `CV_Tatiana_Munoz.pdf` en la carpeta
2. En `index.html` busca: `href="#contact"`
3. Cambia a: `href="CV_Tatiana_Munoz.pdf" download`

### Cambiar colores

En `styles.css` al inicio, cambia las variables:
```css
--primary-color: #0066CC;  /* Tu color principal */
--secondary-color: #00B4D8; /* Tu color secundario */
```
